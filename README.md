# moviesearch

Moviesearch is a Python library for searching for movies.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install moviesearch.

```bash
pip install moviesearch
```

## Usage

```python
import moviesearch
...
```
